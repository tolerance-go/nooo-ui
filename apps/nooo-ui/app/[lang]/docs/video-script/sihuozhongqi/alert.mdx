# 初始化&安装

首先新建一个空文件夹，执行 npx create-next-app@latest --experimental-app 命令，用 next@13 进行项目初始化

下一步我们来集成 tailwindcss 到当前项目环境中

首先安装依赖

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init

然后添加 tailwind 到 postcss 配置中

module.exports = {
plugins: {
tailwindcss: {},
autoprefixer: {},
}
}

最后配置一下 tailwind 自身的配置，指定模板扫描的目录

module.exports = {
content: ["./src/**/*.{html,js}"],
theme: {
extend: {},
},
plugins: [],
}

不要着急，其实 tailwindcss 就是一个 class 类名库，就那么简单啦，不过这些类名是精心设计过的，用过都说好，23 年 1 月 tailwindcss 已经超过 bootstrap
成为 npm 下载量最大的 css 框架了，推荐大家了解一下

tailwindcss 可以在编译阶段使用，作为 postcss 的插件来用，他可以根据项目中使用到的 class 按需生成 css 文件，所以要提供文件扫描范围

autoprefixer 可以根据目标浏览器运行环境，自动添加 css 属性前缀，简单来说就是让我们的页面可以在更多的浏览器中完整访问，样式不受影响
