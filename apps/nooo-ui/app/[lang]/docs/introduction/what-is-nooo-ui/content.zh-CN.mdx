# 什么是 Nooo-UI？

`nooo-ui` 是一款完全开源的基于 `tailwindcss` 的在线设计集散地。你可以在这里找到丰富的设计资源，并很容易集成到大多数你正在使用的开发框架中。

## 动机

我们爱死 `tailwindcss` 了，令人激动的是，截止 2023 年 1 月份，它已经超过 `Bootstrap` 成为 `npm` 下载量最大的 `css` 框架。

随着使用者的数量增多，有越来越多的设计沉淀，我们希望将他们尽可能的收集，处理，展示分享。其他用户可以用极低的成本找到他们需要的这些`模板`，从中汲取灵感同时进一步提升开发效率。

如果刚好你也这么想，我们非常期待你的 PR！

## 什么是 tailwindcss？

> 快速建立现代网站，而不需要离开你的 HTML。

> `tailwindcss` 是一个工具类优先的 CSS 框架，比如 `flex, pt-4, text-center` 和 `rotate-90`，用他们可以直接在您的标记语言中构建任何设计

以上是来自[官网](https://tailwindcss.com/)的作者介绍，其他还能找到一些线上[真实案例](https://tailwindcss.com/showcase)，以及一个[在线编辑器](https://play.tailwindcss.com/)，拥有及时预览和分享功能，当然还有很多其他有意思的内容，比如 JIT，主题，响应式等。
