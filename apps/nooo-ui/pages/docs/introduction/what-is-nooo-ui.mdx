# 什么是 Nooo-UI？

`nooo-ui` 是一款完全开源的基于 `tailwindcss` 的在线设计集散地。你可以在这里找到丰富的设计资源，并很容易集成到大多数你正在使用的开发框架中。

<br />

# 动机

我们爱死了 `tailwindcss`，截止 2023 年 1 月份，它已经超过 `Bootstrap` 成为 `npm` 下载量最大的 `css` 框架，毋庸置疑，`tailwindcss` 受到广大群众的一致好评。

随着使用者的数量增多，有越来越多的设计沉淀，我们希望将他们尽可能的收集，处理，展示。其他用户可以用极低的成本找到他们需要的这些`模板`，从中汲取灵感同时进一步提升开发效率。

如果刚好你也这么想，我们非常期待你的 PR，Salute！

<br />
